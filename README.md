# easychat
A basic chat with socket.io

# live demo
https://easy-chat-web-app.herokuapp.com/

# deploy on heroku
- create new app
- login to heroku cli ("heroku login")
- "heroku features:enable http-session-affinity -a NAME_OF_THE_APP"
- deploy
